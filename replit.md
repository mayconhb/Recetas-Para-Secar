# Recetas para Secar

## Overview
A Spanish-language landing page for a recipe/diet product called "Recetas para Secar" (Recipes to Slim Down). This is a static HTML website with embedded CSS and JavaScript.

## Project Structure
- `index.html` - Main landing page with all content, styles, and scripts embedded
- `README.md` - Original project readme

## Running the Project
The site is served using Python's built-in HTTP server on port 5000.

## Deployment
Configured for static deployment, serving all files from the root directory.
